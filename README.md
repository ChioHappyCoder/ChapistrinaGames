## 🌟 ¿Qué es Chapistrina Games?

Chapistrina Games es una experiencia visual inmersiva diseñada con una estética de **interfaz de videojuego (HUD)**, pero con un propósito educativo. Aquí, los niños de preescolar pueden descubrir mágicos títulos interactivos para dar sus primeros pasos en la lectura y escritura.

### 🕹️ Juegos Destacados

*   **🅰️ El Rescate de las Vocales** *(Nivel 1)* - Ayuda a la mascota a encontrar las vocales perdidas en el bosque mágico.
*   **✨ Palabras Mágicas** *(Nuevo)* - Forma tus primeras palabras y descubre los hechizos ocultos.
*   **📖 Cuentos Interactivos** *(Avanzado)* - Lee, escucha y decide el final de estas increíbles historias animadas.

---

## 🎨 Características del Diseño (¡Puro Poder!)

*   **HUD Interactivo:** Una barra de navegación con estilo de videojuego que incluye nivel de usuario, puntos de experiencia (EXP) y tu inventario (mochila de juegos).
*   **Modo Neón / Glassmorphism:** Interfaz oscura (Dark Mode) con gradientes, desenfoques de cristal y destellos en colores vibrantes (Cian, Rosa Neón, Morado, Verde y Amarillo).
*   **Efectos Dinámicos:** Tarjetas flotantes y botones que reaccionan de manera genial usando CSS puro y Vanilla JS.
*   **100% Responsivo:** Listo para jugarse (o navegarse) desde cualquier PC, tablet o dispositivo móvil.

## 🚀 Cómo inicializar la aventura

Solo necesitas tu navegador favorito para ejecutar el código fuente. ¡Sin bases de datos complejas ni instalaciones raras!

1. Clona el repositorio a tu equipo:
   ```bash
   git clone https://github.com/ChioHappyCoder/ChapistrinaGames.git
   ```
2. Abre la carpeta del proyecto.
3. Haz doble clic en el archivo `index.html` para abrirlo en tu navegador.
4. ¡Presiona Start y comienza a explorar!
